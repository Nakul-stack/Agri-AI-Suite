============================================================
API ENDPOINT:
============================================================

POST /api/fertilizer/recommend

Required Fields:
{
    "N": 37,              # Current soil Nitrogen
    "P": 0,               # Current soil Phosphorus
    "K": 0,               # Current soil Potassium
    "temperature": 26,    # Temperature in Celsius
    "humidity": 52        # Humidity percentage
}

Optional Fields:
{
    "moisture": 38,       # Soil moisture percentage
    "soil_type": "Sandy", # Sandy, Loamy, Clayey, Black, Red
    "crop_type": "Maize", # Crop type
    "ph": 6.5             # Soil pH
}

Response Includes:
- Recommended fertilizer name
- Confidence score
- Top 3 fertilizer recommendations
- Fertilizer composition and information
- Application guidelines
- Dosage estimates
- Organic alternatives
- Soil health notes
